# Sistema de Gestión Escolar (CRUD en Python)

Este proyecto es un sistema de gestión escolar desarrollado en **Python**, que permite administrar estudiantes y trabajadores mediante un menú por consola.

Fue creado como práctica de **Programación Orientada a Objetos (POO)** y operaciones **CRUD**, con el objetivo de fortalecer bases para proyectos reales y portafolio profesional.

---

##  Funcionalidades

###  Estudiantes
- Agregar estudiante
- Mostrar estudiantes registrados
- Actualizar estudiante por ID
- Eliminar estudiante por ID

###  Trabajadores
- Agregar trabajador
- Mostrar trabajadores registrados
- Actualizar trabajador por ID
- Eliminar trabajador por ID

---

## Conceptos aplicados

- Programación Orientada a Objetos (POO)
- Herencia
- Polimorfismo (método `mostrar_datos`)
- Listas
- Menús interactivos por consola
- Identificación de registros mediante ID
- Buenas prácticas en nombres de clases y métodos

---

##  Tecnologías utilizadas

- Python 3.x
- Consola / Terminal
